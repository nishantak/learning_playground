Model

A single neuron is used:

y = tanh(wx + b)

The parameters w and b control the shape and position of the curve.

Target Function

Target function:

y = 2x is a linear function.

The model uses a nonlinear activation (tanh), which can approximate the linear function near the center but saturates at larger values.

This demonstrates how neural networks approximate functions and where limitations appear.

Gradient Estimation

Gradients are estimated numerically using finite differences:

∂L/∂w ≈ (L(w + ε) − L(w)) / ε
∂L/∂b ≈ (L(b + ε) − L(b)) / ε

This approximates how the loss changes when parameters are slightly adjusted.

Learning Rate

The learning rate (η) controls how large each update step is:

w ← w − η ∂L/∂w
b ← b − η ∂L/∂b

Small η → slow but stable learning
Large η → faster but may overshoot

Training Process

Training process:

  1. The model makes predictions
  2. The error (loss) is computed
  3. Gradients estimate how to reduce error
  4. Parameters are updated
  5. Process repeats until the model improves
